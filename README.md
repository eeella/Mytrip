# Mytrip — 濟州島冬季小資六日

冰雪漢拏山冬攀、山茶花與橘子秘境的自駕／公車行程手冊。
單一 HTML 檔，可直接雙擊離線開啟（景點圖片與地圖需連線）。

## 線上預覽
https://eeella.github.io/Mytrip/

## 內容
- 📅 Day 1–6 完整行程（時間、地圖定位、實景照片）
- 🗺️ 互動地圖（依天數分色、住宿據點、可點圖例放大）
- 🧭 交通與住宿方案切換：🚗 自駕版／🚗🚌 混合版／🚌 公車版（含公車號碼班次）

## 檔案
- `JEJU.html` — 行程主檔
- `index.html` — 自動導向主檔（供預覽網址使用）

## 更新方式
編輯後推送即自動更新線上預覽：
```
git add -A && git commit -m "update" && git push
```
